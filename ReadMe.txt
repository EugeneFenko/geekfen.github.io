GeekFen - HTML template!

Main folder for template.
Include other folder and *.html, *.php, *.svg or other type files.

1) css - folder for stylesheets;
2) img - folder for images;
3) js - folder for scripts;
4) video - folder for videos;

5) favicon.svg - site minipicture int title;
6) index.html - default version template;
7) LICENSE - license file;
8) VideoBackground.html - template with videobackground; 
9) YouTubeBackground.html - template with YouTube background;

For install template, copy All file with this folder in your Web server.

For install "Video Background" or "YouTube Background" version, rename "VideoBackground.html" or "YouTubeBackground.html" to "index.html". 

Good Luck!
